News Summarizer is a full-stack application that scrapes and condenses news articles from any URL using FastAPI and React. It features a robust Selenium & BeautifulSoup pipeline to successfully extract text from dynamic, anti-bot protected websites. The core logic utilizes BERT embeddings and K-Means clustering to generate accurate extractive summaries while preserving context. Performance is optimized with a SQLAlchemy caching layer that stores results to eliminate redundant processing and reduce latency
